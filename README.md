#Stone Paper Scissors Game(Java)
This is a simple Stone Paper Scissors game written in Java.
##Features:-
- User vs Computer
- Random computer choices
- Console based game
## How to Run:-
1. Compile the Java file
2. Run the program
3. Enter:-
   - 0 for Stone
   - 1 for Paper
   - 2 for Scissors
